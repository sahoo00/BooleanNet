File descriptions

stepminer-1.0.jar

The StepMiner tool in Java archive format (.jar).

stepminer, stepminer.bat

Scripts to run StepMiner. For unix systems use stepminer and for Windows systems use stepminer.bat.

images/*

The StepMiner icons.

doc/*

The StepMiner documentations in docbook SGML format.

gene_association.goa_human, gene_association.sgd

These are gene annotations files downloaded from http://www.geneontology.org for human and yeast respectively. The gene annotations are used for the "GO analysis" functionality in StepMiner.

gene_ontology.obo

The gene ontology in OBO format. This is also downloaded from http://www.geneontology.org . The gene ontology is used for the "GO analysis" functionality in StepMiner.

yeast.pcl

Yeast time course data from Brauer, et al.

Matthew J. Brauer, Alok J. Saldanha, Kara Dolinski, and David Botstein
